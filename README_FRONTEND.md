This frontend is built with Vite and connects to the backend via `VITE_API_URL` (set in `.env.production`). The live site is on Netlify: https://legendary-starburst-b94dc0.netlify.app

Setup
-----
1. Install dependencies: `npm install`
2. Set environment variable in `.env` or `.env.production`: `VITE_API_URL=https://mbale-innovators-hub-backend-1.onrender.com/api`
3. Run locally: `npm run dev`

- See `LINKS.md` in backend repo for test account credentials and instructions.
